# 添加蓝天白云

<demo html="demo/08.html" title="example" description="添加蓝天白云"></demo>

## 实际上就是给场景添加一个 Cube 背景图

```ts
//添加蓝天白云
const skybox = new THREE.CubeTextureLoader()
  .setPath("demo/skybox/")
  .load(["px.png", "nx.png", "py.png", "ny.png", "pz.png", "nz.png"]);
viewer.scene.background = skybox;
```
