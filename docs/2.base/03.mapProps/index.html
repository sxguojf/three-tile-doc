3. TileMap 属性 ​

TileMap 继承于 threejs 的 Object3D ，除了 Object3D 的属性外，它还有如下属性：

名称类型说明
namestring取得或设置模型名称，默认为'map'
imgSourceISource | ISource[]取得或设置影像数据，必填参数源
demSourceISource | undefined取得或设置地形数据，默认为 undefined 源
LODThresholdnumber设置瓦片 LOD 阈值系数。TileMap 本质上是一个动态 LOD 模型，它根据瓦片离摄像机的距离对模型进行细化或合并，该值越大瓦片细化越快，越小瓦片合并越快，默认为 1
rootTileTile取得根瓦片
loaderITileLoader取得或设置地图瓦片加载器，默认为内置瓦片加载器
minLevelnumber取得或设置地图最小缩放级别，小于这个级别瓦片树不再更新，默认为 2
lon0ProjectCenterLongitude取得或设置中央子午线经度，中央子午线决定了地图的投影中心经度，可设置为-90，0，90，默认为 0
bounds[number, number, number, number]取得地图显示范围，默认为[-180, -90, 180, 90]，即世界范围
projectionIProjection取得地图投影对象，默认为 EPSG:3857 投影
autoUpdateboolean取得或设置瓦片是否在每帧渲染时自动更新，默认为 true
downloadingnumber取得当前正在下载的瓦片数量
